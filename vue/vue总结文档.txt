1:vue.js:执行顺序是先执行html再执行vue.js
2:先执行父辈组件 再去调用子类组件
3:html执行的表达式与html的结合 和  new Vue()执行的代码 不能分开，可以看作同一个执行方法;
        如:下例子 new Vue()中data参数 遍历在blog-post标签里 和遍历在vue()对象是一样的;
	       <div id="blog-post-demo">
				<blog-post v-for="post in posts"  v-bind:key="post.id"  v-bind:title="post.title"></blog-post>
			</div>		
			<script type="text/javascript">
				//debugger
				Vue.component('blog-post', {
					props: ['title'],
					template: '<h3>{{title}}</h3>'
				})
			</script>
			
			<script type="text/javascript">
				var vm=new Vue({
					
					  el: '#blog-post-demo',
					  data: {
					    posts: [
					      { id: 1, title: 'My journey with Vue' },
					      { id: 2, title: 'Blogging with Vue' },
					      { id: 3, title: 'Why Vue is so fun' }
					    ]
					  }
				})
			</script>		
						
4.组件 执行等于vue把对应html拿过来生成一个vue.js 的DOM对象，然后再处理DOM对象;
子组件和父组件如同父子关系分开没直接联系,父组件通过v-bind渲染数据给子组件的对应的html,子组件通过prop向对应的HTML获取数据;
